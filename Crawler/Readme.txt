#class Crawler
crawls public repos of github for a certain language (here is java, but can be chnaged to crawl any other language). The class need an access token in order to work. The crawled repos are saved then in files locally 

# class indexer
use the files saved locally to create an index (json file saved locally). It uses javalang library for parsing java files, in order to parse for  other languages you need a different parser.
